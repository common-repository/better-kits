== Description ==

Better Template Kits is an elementor templates to build your business web site by Page Builder.

*Note:* This plugin is a templates of [Elementor Page Builder](https://wordpress.org/plugins/elementor/) and will only work with Elementor Page Builder installed.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
